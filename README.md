# Phishing Awareness Training

## Description
This project is part of the CodeAlpha Cyber Security Internship.

It explains phishing attacks and how users can protect themselves.

## Topics Covered
- What is phishing
- Types of phishing
- Signs of phishing attacks
- Prevention techniques

## Objective
To educate people about cybersecurity threats and safe online practices.
